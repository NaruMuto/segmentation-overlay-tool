# segmentation-overlay-tool
A tool to generate segmentation overlays using trained models
学習済みの深層学習モデルを使用して、動画ファイルにセグメンテーション結果を重ね合わせた（オーバーレイ）動画を、Google colab上で生成するツールです。
Google ドライブ上の特定のフォルダに「学習済みモデル」と「解析したい動画」を保存して実行することで、AIによる部位識別結果を色付けした動画を出力します。
This tool generates segmentation overlay videos on Google Colab using trained deep learning models.
By simply saving a "trained model" and a "target video" in a specific Google Drive folder and running the script, it outputs a video with color-coded AI segmentation results.
